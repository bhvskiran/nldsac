Write a Program to Check if a Number is Positive , Negative or zero .

Example 1:

Input: number = 23 
Output: positive
Explanation: 23 is a positive number
Example 2:

Input: number = -15 
Output: negative
Explanation: -15 is a Negative number
Example 3:

Input: number = 0 
Output: zero
Explanation: 
Constraints:

10^-4 <= number <= 10^4
